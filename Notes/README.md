# 🐢 Move the Turtle — Course Notes & Lesson Plans

Welcome, coder! 👋 These notes go with the **Move the Turtle** course (CrEAST).
Students learn **Python 3** by drawing pictures, making animations, and building games
with a friendly turtle. No boring theory — you type, you run, you see magic. ✨

Each session is a **full 3-hour lesson plan** with timed blocks, deep explanations,
multiple **hands-on labs**, challenges, and take-home assignments.

---

## 📚 The 7 Sessions (≈3 hours each)

| # | Session | Big ideas | You'll make |
|---|---------|-----------|-------------|
| 1 | [Meet the Computer & Your First Turtle](session-1.md) | input/output, `print`, variables, `input`, turtle basics | Shapes & a happy face 🙂 |
| 2 | [Pen Magic & Loops](session-2.md) | pen control, fill, the `for` loop | Smiley + loop patterns 🖍️ |
| 3 | [Data Types, Operators & If/Else](session-3.md) | int/float/string, operators, booleans, `if/elif/else` | Grade checker + traffic light 🚦 |
| 4 | [Loops Deep Dive — `for` & `while`](session-4.md) | `range`, lists, accumulators, `while`, nested loops | Times tables + the Sun ☀️ |
| 5 | [The Turtle Map, Functions & Animation](session-5.md) | coordinates, screen, **functions (`def`)**, parameters | Olympic rings + a House 🏠 |
| 6 | [Randomness, Animation & Final Project](session-6.md) | `random`, `while True`, `tracer/update`, game logic | Fireworks + moon sky 🎆🌙 |
| 7 | [The Racing Game & Capstone](session-7.md) | multiple turtles, building big programs, your own project | 4-turtle race + capstone 🏁 |

## 🧭 What's inside every session
- 🎯 **Learning objectives** + a **3-hour timed agenda**
- 📖 **Concept blocks** with kid-friendly analogies and "why it works"
- 💻 **Hands-on Labs** with checkpoints (students code the whole time)
- 🔮 **Predict-the-output** boxes — guess first, then click to reveal
- 🐞 **Debugging Corner** — the exact mistakes beginners make, and the fix
- 🏆 **Challenge** + 🏠 **take-home assignment** (with hidden answer keys)
- 🧑‍🏫 **Facilitator notes** — timing, differentiation, what to watch for

## 💻 Where to write code
- Online (easiest): **[trinket.io/python](https://trinket.io/python)** or **[replit.com](https://replit.com)** — both run Python 3 with turtle.
- On a computer: save a file like `square.py` and run `python3 square.py`.

> 🐢 **The Golden Rule:** every drawing program ends with `turtle.done()` — or the
> window closes before you can see your art!

## 🧑‍🏫 For facilitators / interns
- Each session is paced for **3 hours** with two lab blocks and a break. If you have less
  time, the **Challenge** is the cut-first item; never cut Lab A.
- Keep a **running pace**: aim for students to be *typing and running code* at least half
  the session.
- **Functions (`def`)** are introduced in **Session 5** — make sure earlier sessions stay
  on shapes/loops/decisions so the leap lands well.
- Mark the **mini-test (S5)** and **final test (S6)** with the rubrics in those files.
- Absent students should complete missed labs before the tests/game.
