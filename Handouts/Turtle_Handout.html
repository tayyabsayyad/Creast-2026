Every turtle program starts the same way. Type these two lines first:

import turtle t = turtle.Turtle() # t is your pen / turtle

Move & Turn

CommandWhat it does
t.forward(100)go forward 100 steps
t.backward(100)go back 100 steps
t.right(90)turn right 90°
t.left(90)turn left 90°
t.goto(x, y)jump to a point
t.setheading(0)face a direction

Pen up / down

CommandWhat it does
t.penup()lift pen — move, no line
t.pendown()drop pen — draw again
t.pensize(5)make the line thick
t.speed(5)0 = fastest, 10 = fast

Colour & Fill

CommandWhat it does
t.color("red")pen colour
t.fillcolor("yellow")colour inside a shape
t.begin_fill()start colouring inside
t.end_fill()stop colouring inside
t.shape("turtle")turtle / arrow / circle

Shapes & Finish

CommandWhat it does
t.circle(50)draw a circle, radius 50
t.dot(20)draw a filled dot
t.write("Hi")write text
t.hideturtle()hide the turtle
turtle.done()last line — keep window open

Drawing a square — read it out loud

import turtle t = turtle.Turtle() for i in range(4): # do it 4 times t.forward(100) # one side t.left(90) # one corner turtle.done()
Notice 👀
• range(4) = 4 sides  →  change to range(3) for a triangle (and turn 120).
• The two lines under for are indented (pushed in). That tells Python they belong to the loop.
• Try changing 100, the colour, or the turn angle. Experiment!

🗺️ The Turtle Map & Loops

Where is the turtle? How do we repeat things?

Page 2 of 2

The screen is a map — the middle is (0, 0)

  • Middle of the screen is (0, 0).
  • Go right → x gets bigger (+). Go left → x gets smaller (−).
  • Go up → y gets bigger (+). Go down → y gets smaller (−).
  • t.goto(100, 100) jumps to the green dot.
  • Use t.penup() before goto if you don't want a line while jumping.
Try it: where is t.goto(-100, -100)?   (Answer: bottom-left!)

Two ways to repeat — Loops

# for loop: when you know HOW MANY times for i in range(5): print("Hello") # prints Hello 5 times
# while loop: repeat UNTIL something changes punch = 1 while punch <= 3: print("punch!") punch = punch + 1
