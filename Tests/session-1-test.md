**1. What is the correct order of how a computer works?**
- A) Output → Process → Input
- B) Input → Process → Output
- C) Process → Input → Output
- D) Input → Output → Process

**2. Computers only understand which kind of numbers?**
- A) 1 to 10
- B) Odd numbers
- C) 0s and 1s (binary)
- D) Decimal numbers

**3. Why do we use a programming language like Python?**
- A) Because computers love English
- B) It is easy for humans to read and the computer can follow it
- C) It makes the computer slower
- D) Because binary is illegal

**4. What does this print?**
```python
print("Hello, world!")
```
- A) `Hello, world!`
- B) `print("Hello, world!")`
- C) `Hello world`
- D) Nothing

**5. What is the `#` symbol used for in Python?**
- A) To multiply numbers
- B) To write a comment (a note Python ignores)
- C) To end the program
- D) To print text

**6. What is the output of this code?**
```python
num1 = 15
num2 = 6
print("sum =", num1 + num2)
```
- A) `sum = 156`
- B) `sum = num1 + num2`
- C) `sum = 21`
- D) `sum = 9`

**7. What type of value does `input()` always give back?**
- A) A whole number
- B) A decimal
- C) Text (a string)
- D) True or False

**8. What does this print if the user types `5` and `3`?**
```python

y = input("Number: ")
print(x + y)
```
- A) `8`
- B) `53`
- C) `15`
- D) Error

**10. What does `t.left(90)` do?**

- D) Changes the colour to 90
